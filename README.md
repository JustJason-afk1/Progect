Продвинутый рандомайзер по процентам выпадения — это система случайного выбора элементов, в которой вероятность получения каждого результата задаётся в процентах. В отличие от обычного случайного выбора, здесь учитывается вес (шанс) каждого элемента, что позволяет гибко управлять частотой их выпадения.
Инструкция по запуску: 
1. установить python
2. запустить `runner.py`
